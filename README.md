# Cars Dealership Web Application

## Repository Name: `expressBookReviews`

### Project Overview
The **Cars Dealership Web Application** is a full-stack project designed to allow users to view dealership branches, add and view car reviews, and interact with car models and makes. The application is built using React for the frontend, Django for the backend, and integrates with multiple technologies like MongoDB, SQLite, and Kubernetes for deployment. 

This application also leverages Docker for containerization, and IBM Cloud Code Engine for deployment. 

---

## Technologies Used:
- **Frontend**: React.js, HTML, CSS
- **Backend**: Django, Node.js, Express.js
- **Database**: MongoDB, SQLite
- **Containerization & Deployment**: Docker, Kubernetes, IBM Cloud Code Engine
- **CI/CD**: GitHub Actions
- **Others**: Flask for microservices, Sentiment Analysis Integration
